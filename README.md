# Stellar Wallet

Desktop / Mobile app, payment requests, multisignature and more.


## Setup

```
npm install
```

## Development

```
npm run dev
```

To run the tests:

```
npm test
```

To run the storybook:

```
npm run storybook
```

## Build

```
npm run build
```
